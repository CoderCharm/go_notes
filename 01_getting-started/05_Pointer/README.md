## 指针和地址
> 上来咱就讲指针和地址，go各种数据类型，字符串，数组，map看看语法，多敲敲

可以参考下知乎问题[为什么要有指针](https://www.zhihu.com/question/26623283)

我现在的理解就是， 指针通俗来讲就是内存地址，直接通过内存地址修改某个值，这是最直接，也是效率最高的。
